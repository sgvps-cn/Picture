# 图片上传前端页面

一个简洁、美观且功能完整的图片上传前端页面，支持切换不同的上传接口（360/搜狗），并提供友好的用户交互和状态反馈。

![页面预览](https://img.shields.io/badge/页面状态-可用-brightgreen)
![技术栈](https://img.shields.io/badge/技术栈-HTML%20%7C%20CSS%20%7C%20JavaScript-blue)
![响应式](https://img.shields.io/badge/适配-移动端%20%7C%20PC端-orange)

## 🌟 功能特性

- 📤 支持图片文件上传（仅允许图片格式）
- 🔄 可切换不同的上传接口（360接口/搜狗接口）
- 💾 记住用户上次选择的接口（基于localStorage）
- 🎨 美观的UI设计，包含阴影、圆角、悬停效果
- 📱 完全响应式布局，适配移动端和PC端
- 📋 清晰的上传状态反馈（成功/失败提示）
- ⚡ 简洁的代码结构，易于扩展和维护
